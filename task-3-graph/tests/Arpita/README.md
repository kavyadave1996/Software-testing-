# Phase 3: Graph-Based Test Coverage – HTTPie Project

## 🎯 Objective

This phase evaluates and improves test coverage for selected components of the HTTPie CLI project using **graph coverage criteria**. The goal is to:
- Measure current test suite coverage
- Identify partially covered files
- Extend the test suite to improve coverage
- Achieve 100% coverage for at least one key file

---

## 🧪 Tools Used

| Tool        | Purpose                          |
|-------------|----------------------------------|
| `pytest`    | For running unit tests           |
| `coverage.py` | To measure statement and branch coverage |
| `pytest-cov`| Integration of coverage with pytest |
